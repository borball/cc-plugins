Show details of a Red Hat support case. Arguments: $ARGUMENTS

Run:
```bash
CLAUDE_PLUGIN_DATA="${CLAUDE_PLUGIN_DATA}" ${CLAUDE_PLUGIN_ROOT}/scripts/rh-case-show.sh $ARGUMENTS
```

Options: `--comments` to include comments, `--attachments` to list attachments.
